Built a Movie Explorer web application that fetches movie data from a REST API.  

Users can:
- Browse a list of movies with posters, release dates, and IMDb ratings  
- Search movies in real-time using the search input  
- Sort movies by rating (low → high / high → low)  
- View detailed information about each movie in a modal window  
- Add and remove movies from a favorites list  

Favorites are stored using LocalStorage (set/get) to persist user data in the browser.  

This project demonstrates working with API integration, DOM manipulation, event handling, filtering, sorting, and client-side data storage.
